## ADDED Requirements

### Requirement: 语音合成模型选择器

系统 SHALL 在项目设置模态框中提供语音合成模型（audioModel）选择器，与其他模型选择器保持一致的 UI 风格。

#### Scenario: 显示语音合成模型选项

- **WHEN** 项目设置模态框打开
- **THEN** 显示语音合成模型选择器，列出用户已配置的所有语音合成模型

#### Scenario: 选择语音合成模型

- **WHEN** 用户从选择器中选择一个语音合成模型
- **THEN** 系统更新项目配置中的 audioModel 字段

#### Scenario: 模型能力参数配置

- **WHEN** 选中的语音合成模型有能力参数（如音质、语速等）
- **THEN** 显示对应的能力参数选择器

### Requirement: 旁白音色配置区块

系统 SHALL 在项目设置模态框中提供旁白音色配置区块，支持 AI 设计音色功能。

#### Scenario: 显示旁白音色配置区块

- **WHEN** 项目设置模态框打开
- **THEN** 显示旁白音色配置区块，包含当前音色状态和操作按钮

#### Scenario: 旁白音色配置区块与模型选择器联动

- **WHEN** 用户选择了语音合成模型
- **THEN** 旁白音色配置区块启用，允许配置音色
